## 🎯 Platform Overview

**HAWKMOTH** is a professional development platform that combines conversational interfaces with Git integration, automated deployment, and self-improvement capabilities. Named for precision and focus, HAWKMOTH represents the evolution from experimental tool to production-ready development platform.

### **Design Philosophy:** 
Clean, modern, professional interface with hawkmoth-inspired branding elements. Focus on functionality, clarity, and efficient development workflows.

## 🏗️ Technical Architecture

### **Core Components:**
```
HAWKMOTH-Project/
├── app.py                    # FastAPI backend with Git integration
├── frontend.html             # Clean modern interface (system fonts)
├── conversation.py           # Rule-based chat system with HAWKMOTH commands
├── git_handler.py           # HAWKMOTHGitHandler for repository management

```

### **Development Workflow:**
- **Summarize → Confirm → Commit** - All changes follow approval workflow
- **Iteration-based Development** - `/working/` directory for code iterations
- **Git Integration** - Automatic versioning and deployment management
- **Session Management** - Chat capacity monitoring and transition protocols

### **Green/Blue Deployment System:**
```
Production Environment (HAWKMOTH v0.0.0)
    ↓ User: "improve hawkmoth"

```
